✂️ Data Splitting - The Golden Rule ▼

🚨 THE GOLDEN RULE

NEVER LOOK AT TEST DATA UNTIL THE VERY END!

🎲 train_test_split() - Random Split
from sklearn.model_selection import train_test_split train_set, test_set = train_test_split( housing, test_size=0.2, random_state=42 )
Parameter Purpose Example
test_size=0.2 80% train, 20% test Standard split
random_state=42 Reproducible results Same split every time
stratify=y Keep proportions Balanced classes
🎯 StratifiedShuffleSplit - Smart Sampling
from sklearn.model_selection import StratifiedShuffleSplit split = StratifiedShuffleSplit( n_splits=1, test_size=0.2, random_state=42 )
🧠 Why Stratified?
If your target has categories, this ensures your train/test sets represent them proportionally!

⚖️ Feature Scaling - Level the Playing Field
Scaler Formula Best For
StandardScaler (x - mean) / std Neural networks, SVM
MinMaxScaler (x - min) / (max - min) When you need 0-1 range
🚨 Critical: Always fit scalers on training data only!
